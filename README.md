## 🏡 Airbnb Market Analysis Dashboard##

## 📌 Project Overview##

This project analyzes Airbnb listing data to uncover insights related to pricing trends, host behavior, geographic distribution, and guest engagement.
Interactive dashboards were built using Tableau to visualize key patterns and support data-driven decision making.

## 📊 Key Insights##

Manhattan and Brooklyn have the highest number of listings and revenue contribution.

Entire home/apartment listings have the highest prices compared to private or shared rooms.

Booking demand shows seasonal variations across months.

Hosts with multiple listings dominate a large portion of the market.

Instant bookable listings tend to attract more guest engagement.

## 📈 Dashboard Analysis##

The project includes multiple dashboards focusing on:

Executive Summary – Overall market metrics and listing distribution

Geographic Insights – Listing distribution and revenue by neighborhood

Host Behavior Analysis – Host listing patterns and engagement trends

Seasonal Pricing Trends – Monthly booking and price variations

Guest Reviews Analysis – Ratings and feedback across locations

Room Type & Policy Trends – Impact of room types and cancellation policies on pricing
